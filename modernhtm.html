Q1. List out the features of HTML5 Ans. There are different features in HTML5
  1. Semantics Tags - Give meaning to page parts for better structure.

  2. Audio and video - Play sound and videos directly in the browser

  3. Forms input - New input types like email, date, range make forms smarter.

  4. Svg - Draw graphics, charts, games in-browser using JS.

  5. Drag and drop - Drag elements and drop them elsewhere with ease.
